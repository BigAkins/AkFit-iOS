# Visual Rules

AkFit should feel:

- clean
- premium
- modern
- minimal
- fast
- easy to scan
- iOS-native
- macro-first

## Keep
- large bold titles
- soft light card backgrounds
- strong spacing
- rounded corners
- dark primary buttons
- simple icon use
- uncluttered screens
- large touch targets

## Avoid
- overly crowded layouts
- generic wellness clutter
- too many competing metrics on one screen
- noisy gradients
- gimmicky AI visuals
- small hard-to-tap controls

## Product emphasis
AkFit should visually prioritize:
- calories
- protein
- carbs
- fat
- portion clarity
- food logging speed
- daily targets and remaining macros

AkFit should de-prioritize:
- steps
- water
- generic health scores
- excessive motivational content
- exercise-heavy UI in MVP
