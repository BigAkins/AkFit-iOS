# Onboarding Notes

Onboarding should be useful, not bloated.

## Must help the user do these things
- enter body stats
- choose a goal
- choose a target pace
- understand calorie and macro targets

## Onboarding should not
- feel too long
- ask too many marketing questions
- overload the user with persuasion screens
- delay getting into the actual product

## AkFit onboarding tone
- clear
- confident
- practical
- performance-focused
- low fluff

## Best patterns to keep
- large question headers
- clear card selections
- wheel pickers for body metrics
- simple progress indicator
- strong bottom CTA
