# Dashboard Notes

The dashboard is one of the most important screens in AkFit.

## What the dashboard should do
- show calorie target and remaining calories clearly
- show protein, carbs, and fat clearly
- help the user know what is left for the day
- make logging food feel easy and obvious
- surface recent food activity

## Dashboard priorities
1. calories
2. protein
3. carbs
4. fat
5. recent foods / recent logs
6. quick add actions

## Do not over-prioritize
- steps
- water
- generic wellness widgets
- too many secondary metrics
- exercise-first content

## UI direction
- clean large cards
- easy-to-read macro values
- floating add button is good
- dashboard should feel useful in under 3 seconds
- users should immediately understand whether they are on track
